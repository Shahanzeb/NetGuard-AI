# NetGuard-AI
An AI-Powered Network Intrusion Detection and Automated Mitigation System for my Final Year Project.

 **Project Overview**

NetGuard AI is an intelligent, self-healing security system designed to detect and mitigate network intrusions in real-time. By combining Machine Learning for threat classification and Python-based automation for instant response, this framework bridges the gap between passive monitoring and active defense.

SRS (https://github.com/Shahanzeb/NetGuard-AI/blob/main/NetGaurd%20SRS.docx)

**Core Objectives**

Real-Time Detection:
Sniffing network traffic to identify malicious patterns using the NSL-KDD dataset.


Automated Mitigation:
Automatically reconfiguring virtual Cisco routers to block attackers via SSH/Netmiko.


Centralized Management:
Providing a web-based dashboard for administrators to monitor network health and security logs.

**Tech Stack**

Programming: Python 3.x 


AI/ML: Scikit-learn (Random Forest/SVM), Google Colab 


Network Simulation: GNS3 with Cisco IOU/IOS images 


Automation: Netmiko, Pyshark, Scapy 


Frontend: React.js & Tailwind CSS 


Backend: Flask & MySQL 

**Project Scope**

In-Scope:
ML model training, GNS3 topology integration, automated ACL deployment, and dashboard development.


Out-Scope:
Physical hardware deployment and mobile application versions.
